This project is a pacient simulator made in unity

Para rodar o projeto é preciso utilizar o git lfs pois o pacote de assets é muito grande para ser colocado no git

https://git-lfs.github.com/

Para rodar o projeto é necessario baixar a pasta com o projeto e o pacote de assets "assetscompleto.unitypackage"

Primeiramente a pasta do projeto deve ser baixada e aberta na unity

Em seguida baixe o pacote de assets e na unity vá em Assets > Import Package > Custom Package e abra o pacote de assets

Depois disso vá em File >Open Scene e seleciona a scene SimuladorPaciente para carregar tudo e pronto!
